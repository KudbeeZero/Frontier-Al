### Tailwind Config Fix (REQUIRED)
The `landscape:` Tailwind variant is NOT built-in — it must be declared as a custom screen. Added:

**tailwind.config.ts — Before:**
```ts
theme: {
  extend: {
    borderRadius: {
```

**tailwind.config.ts — After:**
```ts
theme: {
  extend: {
    screens: {
      landscape: { raw: "(orientation: landscape)" },
    },
    borderRadius: {
```

**After:**

## T003 — Fix FlatMap Texture Scrolling (Anti-Banding Formula)

### File
`client/src/components/game/FlatMap.tsx` — drawing block ~lines 354–396

### Root Cause
Previous formula derived `lngFrac`/`latFrac` as `[0,1]` fractions and computed absolute canvas positions. At polar latitudes the vertical offset exceeded the texture height, causing a blank band. The longitude wrapping didn't use modulo so drift could accumulate.

### Change

**Before:**
```ts
const lngFrac = ((cameraRef.current.centerLng + 180) / 360); // [0,1]
const latFrac = ((90 - cameraRef.current.centerLat) / 180);  // [0,1]
const xOff = cx - R - lngFrac * texW + texW / 2;
const yOff = cy - R - latFrac * texH + texH / 2;

const drawWrapped = (img, alpha, op = "source-over") => {
  for (const dx of [-texW, 0, texW]) {
    ctx.drawImage(img, xOff + dx, yOff, texW, texH);
  }
};

// Clouds: cloudXOff = cx - R - (lngFrac * 0.97) * texW + texW / 2
```

**After:**
```ts
const lng = cameraRef.current.centerLng;
const lat = cameraRef.current.centerLat;
const W = R * 2;
const H = R * 2;

// Horizontal: modulo-wrapped so no gap at dateline
let xOff = -(lng / 360) * W;
xOff = ((xOff % W) + W) % W;
xOff = xOff - W;

// Vertical: subtle + clamped to [-R*0.35, +R*0.35] to prevent polar banding
let yOff = (lat / 180) * (R * 0.35);
yOff = Math.max(-R * 0.35, Math.min(R * 0.35, yOff));

const drawWrapped = (img, alpha, op = "source-over", extraX = 0, extraY = 0) => {
  const x0 = cx - R + xOff + extraX;
  const y0 = cy - R + yOff + extraY;
  ctx.drawImage(img, x0,     y0, W, H);
  ctx.drawImage(img, x0 + W, y0, W, H);
  ctx.drawImage(img, x0 - W, y0, W, H);
};

// Clouds: extraX = W * 0.02 for parallax drift
drawWrapped(cloudsImageRef.current, 0.25, "source-over", W * 0.02, 0);
```

### Key Differences
- Longitude uses `((x % W) + W) % W` — always in `[0, W)`, never accumulates drift
- Vertical capped at ±35% of radius — texture always covers the canvas, no blank bands at poles
- Cloud parallax via `extraX` offset rather than separate xOff/yOff computation (cleaner)
