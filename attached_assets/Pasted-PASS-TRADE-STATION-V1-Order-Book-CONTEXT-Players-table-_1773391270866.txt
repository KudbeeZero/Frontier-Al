EDIT 4 — server/storage/db.ts
Implement the four methods above.

fillTradeOrder must be ATOMIC — use a db.transaction() block:
1. Fetch the order, verify status === "open"
2. Fetch offerer player, verify they still have giveAmount of giveResource
3. Fetch filler player, verify they have wantAmount of wantResource
4. Deduct giveResource from offerer, credit wantResource to offerer
5. Deduct wantResource from filler, credit giveResource to filler
6. Set order status = "filled", filledById, filledAt
7. Return success

If any check fails, rollback and return { success: false, error: "..." }

cancelTradeOrder: verify orderId belongs to playerId, set status = "cancelled"
