T003 — FIX FlatMap TEXTURE SCROLLING (FlatMap.tsx)
========================================================
FILE: client/src/components/game/FlatMap.tsx
TARGET: the drawImage block around lines ~355–376 where textures are drawn like:
  ctx.drawImage(img, cx - R, cy - R, R * 2, R * 2)

ROOT CAUSE
- The equirectangular texture is always drawn at the same pixel center, so it never reacts to camera.centerLng/centerLat.
- Plots are recomputed based on camera center, so plots “move” but texture remains static.

IMPLEMENTATION GOAL
- Before drawing EACH texture layer (terrain, night, clouds), apply an offset derived from camera center.
- Draw wrapped copies to handle dateline seam (texture repeats horizontally).

MATH (use these)
- Let W = R * 2, H = R * 2.
- Longitude scroll:
    xOff = -(cameraRef.current.centerLng / 360) * W
  Then wrap xOff to keep it bounded:
    xOff = ((xOff % W) + W) % W
    xOff = xOff - W
- Latitude scroll (approximation):
    yOff = (cameraRef.current.centerLat / 180) * (R * 1.0)
  Keep it subtle; do NOT over-shift vertically.

RENDERING PATTERN (USE THIS EXACT APPROACH)
- Create a small helper inside the render/draw scope:

  const lng = cameraRef.current.centerLng;
  const lat = cameraRef.current.centerLat;
  const W = R * 2;
  const H = R * 2;

  let xOff = -(lng / 360) * W;
  xOff = ((xOff % W) + W) % W;
  xOff = xOff - W;

  const yOff = (lat / 180) * (R * 1.0);

  const drawWrapped = (img: HTMLImageElement, extraX = 0, extraY = 0) => {
    const x0 = cx - R + xOff + extraX;
    const y0 = cy - R + yOff + extraY;

    ctx.drawImage(img, x0, y0, W, H);
    ctx.drawImage(img, x0 + W, y0, W, H);
    ctx.drawImage(img, x0 - W, y0, W, H);
  };

- Replace each layer’s fixed drawImage call with drawWrapped:
  - Terrain: drawWrapped(mapImageRef.current)
  - Night: drawWrapped(nightImageRef.current)
  - Clouds: drawWrapped(cloudsImageRef.current, W * 0.02, 0)  // slight parallax

- Preserve existing alpha and composite operations.
- Do NOT change plot math in this task — only texture drawing.

ACCEPTANCE CHECK
- Dragging left/right: terrain texture scrolls with plots (no longer “stuck”).
- Seam at dateline does not visibly tear (wrapped draws cover it).
- Clouds move similarly, slightly offset for depth.
